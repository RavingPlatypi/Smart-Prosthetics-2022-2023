# Varied Pressure Code

## Description

This is a FULLY working code, where the vibration motor is varied based off off the pressure applied onto the force pressure sensor.

## Usage 
Must need:

- Arduino Uno Board
- 4 A101 Force Pressure Sensors
- 4 Vibration Motors
- 4 100kOhm Resistors
